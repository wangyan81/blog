Git贮藏

当需要checkout出新的分支又不想提交当前改动时，利用贮藏储存然后checkout 新的分支，完成工作后继续上次的工作再应用贮藏的改动。

新增的文件不在git版本控制中不会贮藏，如果需要贮藏可以先git add [file]再进行贮藏
